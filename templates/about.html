Project Scope

This personal development initiative is for a 12-week Code Louisville bootcamp course in Python programming and data analysis.

It consists of a command-line utility, this Flask-based website, and a personal GitHub repository. That repository includes more information as well as a Wiki Page, Kanban board, and project schedule.

This program also includes SQL functions using SQLite3, using the same dataset publicly available from IMDB, and an Exploratory Data Analysis using Jupyter Notebook.

Background

Lacey Chabert is widely regarded as the "queen" of Hallmark movies, having starred in over 30 Hallmark original romantic comedies, dramas, and mysteries. Six Degrees of Kevin Bacon is a meme that cast him as "center of the universe" in Hollywood.

Based on how many movies he's starred and costarred in with top actors and actresses in Hollywood, analysts joked that nobody in Hollywood was more than six degrees of separation from Kevin. And they backed it up with research and analyis. Other studies have proven several people are more likely candidates as the center of the Hollywood metaverse.

This neophyte Python program performs similar analysis on Lacey Chabert within Hallmark movies.

Functionality

You will find the command line utility and webpage have some different functions, as it is not intended to be a full "port" of the application. For example, the utility performs some "fuzzy logic" on actor names and movie titles in a SQL database to find possible matches for a search. The website requires exact spelling.

The utility shows degrees of separation between two people, and movies they starred in together. The webpage only shows degrees of separation to Lacey.

The utility calculates and shows a few of the graphs included in the Jupyter Notebook analysis. The webpage does not.

The webpage includes links to the project pages on GitHub and the IMDB pages for the actor and movie, the utility does not.
